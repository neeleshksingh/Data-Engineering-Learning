# Data Engineering Bootcamp

Welcome to the Data Engineering Bootcamp! This project is designed to provide a comprehensive learning experience in data engineering concepts and practices. Below is an overview of the project structure and its contents.

## Project Structure

- **01_SQL/**: This directory contains subdirectories focused on SQL concepts and techniques.
  - **01_Group_By/**: Exercises and examples related to the SQL GROUP BY clause.
  - **02_Joins/**: Materials on different types of SQL joins (INNER, LEFT, RIGHT, FULL).
  - **03_Window_Functions/**: Focus on SQL window functions and their applications.
  - **04_CTE/**: Coverage of Common Table Expressions (CTEs) and their usage in SQL queries.
  - **05_Date_Problems/**: Challenges and solutions related to date manipulation in SQL.
  - **06_Advanced/**: Advanced SQL topics and techniques.

- **02_Python/**: Python scripts and notebooks related to data engineering tasks.

- **03_PostgreSQL/**: Resources and examples specific to PostgreSQL database management.

- **04_ETL/**: Focus on Extract, Transform, Load (ETL) processes and tools.

- **05_Airflow/**: Materials related to Apache Airflow for workflow orchestration.

- **06_Docker/**: Resources on using Docker for containerization in data engineering.

- **07_Spark/**: Focus on Apache Spark for big data processing.

- **08_Azure/**: Resources related to Azure cloud services for data engineering.

- **09_Projects/**: Various projects that apply the skills learned in the bootcamp.

## Usage Instructions

To get started, navigate to the relevant directory based on the topic you wish to explore. Each section contains exercises, examples, and resources to help you learn and practice data engineering skills.

Happy learning!
